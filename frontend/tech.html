技术架构

深度了解我们的技术

探索 MechanicalAI 背后的技术架构，从大模型到前端交互的完整技术栈

系统概览

整体架构

MechanicalAI 采用现代化的云原生架构设计，确保高性能和高可用性

系统架构图

前端层

  • HTML5 + CSS3 + JavaScript
  • Tailwind CSS
  • Lucide Icons
  • Particle.js (背景动画)

后端层

  • FastAPI (Python)
  • Uvicorn ASGI Server
  • WebSocket 实时通信
  • CORS 跨域支持

AI 模型层

  • DeepSeek LLM API
  • 结构化输出 (JSON)
  • 提示词工程
  • 工程知识库

前端技术栈

用户界面

采用纯原生 HTML/CSS/JS 实现，无需复杂的构建系统，易于部署和维护。

UI 设计

Tailwind CSS 提供快速开发和可定制的组件库，实现玻璃态效果和紫色/粉色主题配色。

图标系统

使用 Lucide 图标库，提供统一、美观的 SVG 图标，支持动态加载。

部署平台

前端部署在 Vercel，享受全球 CDN 和自动 HTTPS 支持。

后端技术栈

Web 框架

FastAPI 是一个高性能的现代 Python Web 框架，支持异步请求处理。

实时通信

WebSocket 协议实现实时评论推送，支持全双工通信。

数据验证

Pydantic 提供类型提示驱动的数据验证和序列化。

部署平台

后端部署在 Hugging Face Spaces，提供 GPU 资源和自动扩展。

AI 系统

大语言模型集成

我们精心设计的提示词工程，让 AI 成为专业的工科导师

结构化输出

使用 DeepSeek API 的 response_format 功能，确保 AI 输出符合我们期望的 JSON 结构。

提示词设计

经过精心调优的系统提示词，指导 AI 以启发式教学方式解释复杂的工科概念。

多阶段推理

AI 不仅回答问题，还提供诊断报告、测试题目和学习计划，形成完整的学习闭环。

开源与贡献

快速开始

1. 克隆仓库
git clone https://github.com/yuansgitlab/MechanicalAI
2. 安装依赖
cd backend
pip install -r requirements.txt
3. 配置环境变量
export DEEPSEEK_API_KEY=your_api_key
4. 运行服务
uvicorn main:app --reload
